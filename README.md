# SkillSwap - Платформа обмена навыками

Фронтенд-часть SPA-приложения SkillSwap - социальной сети для обмена навыками.

## Основная информация

В проекте реализована архитектура FSD

Технический стек: React+TS+CSS, Redux Toolkit, React Router, Vite

## Запуск приложения

1. Установить зависимости:

```
npm i
```

2. Установить и запустить [бэкэнд приложения](https://github.com/albrekhtdurer/skillswap_back)

3. Запустить приложение:

```
npm run dev
```

4. Приложение будет доступно на `http://localhost:5173/`
