# SkillSwap - Платформа обмена навыками

Авторы: Команда 42_5

## Запуск приложения

1. Устанавливаем зависимости:

```
npm i
```

2. Запускаем приложение:

```
npm run dev
```

3. Приложение будет доступно на `http://localhost:5173/`

## Для разработчиков

Перед каждым коммитом запускаем скрипт `npm run check`. Если проверки не проходят, то сначала вносим правки, добавляем их к коммиту, после этого уже коммитим.
